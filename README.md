# Country Info App con React 

**Descripción**: Aplicación React para visualizar y filtrar un listado de países con su bandera, capital y continente. Permite buscar países por nombre desde un input.

**Stack**: `React` + `Vite` + `Sass`.

## Instalación

- Clonar: `git clone <repo>`
- Instalar dependencias: `npm install`

## Estructura principal

- `src/components/App.jsx`: Componente principal; contiene el estado de la lista y el input de búsqueda.
- `src/components/CountryList.jsx`: Itera la lista filtrada y renderiza cada `CountryItem`.
- `src/components/CountryItem.jsx`: Componente que muestra bandera, nombre, capital y continente.
- `src/styles/App.scss`: Estilos globales y anidados bajo `.country-info-app`.
